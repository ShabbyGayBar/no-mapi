# No MAPI

![Steam Subscriptions](https://img.shields.io/steam/subscriptions/3697599937?logo=steam)
![Steam Downloads](https://img.shields.io/steam/downloads/3697599937?logo=steam)

[简体中文](documentation/README_zh-CN.md)

This mod removes the market access price impact (MAPI) mechanics introduced in the 1.5 update by setting the initial local market access impact to **100%**.

With this mod, there will be **no profit loss** from mismatches between the local market price and the global market price.

Local goods, i.e. electricity, services, and transportation, **will** still be determined by the **local market price**, though. If you want them to be circulated in the global market, you can subscribe to the [Remove Local Market](https://steamcommunity.com/sharedfiles/filedetails/?id=3696161239) mod collection.

## Getting Started

You can install the mod via
1. [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3697599937)
   or
2. [Paradox Mod Plaza](https://mods.paradoxplaza.com/mods/139775/Any)
   or
3. [GitHub Releases](https://github.com/ShabbyGayBar/no-mapi/releases)

## Compatibility

Theoretically compatible with every single mod out there.

Please report any incompatibility you encounter in the [Issues tab](https://github.com/ShabbyGayBar/no-mapi/issues) using the bug report template.
