# 移除本地市场

![Steam Subscriptions](https://img.shields.io/steam/subscriptions/3697599937?logo=steam)
![Steam Downloads](https://img.shields.io/steam/downloads/3697599937?logo=steam)

使用此模组后，本地市场价格影响机制将被禁用。也就是说，省份内的商品价格将**不再**受到**本地**市场的影响，而**只**取决于**全国**市场的供需（除非基建不够）。

**本地商品**（如电力、服务和运输）机制将仍然**保持不变**。如果您也希望这些机制被移除，可以订阅[移除本地市场](https://steamcommunity.com/sharedfiles/filedetails/?id=3696161239)模组合集。

## 安装

您可以通过以下方式安装此模组：
1. [Steam创意工坊](https://steamcommunity.com/sharedfiles/filedetails/?id=3697599937)
   或
2. [Paradox Mod Plaza](https://mods.paradoxplaza.com/mods/139775/Any)
   或
3. [GitHub Releases](https://github.com/ShabbyGayBar/no-mapi/releases)
